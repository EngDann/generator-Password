# Password Generator

## English

## Description

The "Password Generator" is a project developed entirely in pure JavaScript, featuring a dynamic user interface for generating passwords. Users can select the desired password length from 5 to 30 characters using a slider. After generating a password, users can click on the password text to copy it to the clipboard. A text field is also provided to verify if the password has been successfully copied.

## Technologies Used

- **JavaScript**: Used for dynamic interaction and advanced DOM manipulation.
- **Sass**: Employed to create a responsive and visually appealing interface.
- **HTML**: Structured with HTML5 to enhance accessibility and web semantics.

## Features

- **Adjustable Password Length**: Users can adjust the password length using a slider.
- **One-Click Copy**: Passwords can be copied with a simple click on the generated password text.
- **Clipboard Verification**: Includes a text field to check if the password has been copied successfully.

## Challenges and Learnings

This project was a great learning experience as it involved intricate DOM manipulations and advanced CSS techniques. Developing the slider for adjusting the password length taught me about responsive design elements and handling user inputs in JavaScript. Implementing clipboard functionality was challenging but rewarding, as it significantly improved the user experience of the application.

## How to Use

To use the "Password Generator", follow these steps:

1. **Clone the repository**:

    ```bash
    git clone https://github.com/EngDann/PasswordGenerator.git
    ```

2. **Access the project folder**:

    ```bash
    cd PasswordGenerator
    ```

3. **Open the project**:

    You can open the project in any web browser of your choice. If using Visual Studio Code, you might use the Live Server extension to view the project live.

4. **Generate a password**:

    Use the slider to select the desired length and then click "Generate". Click on the generated password to copy it to the clipboard.

## Português

## Descrição

O "Gerador de Senhas" é um projeto desenvolvido inteiramente em JavaScript puro, apresentando uma interface de usuário dinâmica para geração de senhas. Os usuários podem selecionar o comprimento desejado da senha de 5 a 30 caracteres usando um controle deslizante. Após gerar uma senha, os usuários podem clicar no texto da senha para copiá-la para a área de transferência. Um campo de texto também é fornecido para verificar se a senha foi copiada com sucesso.

## Tecnologias Utilizadas

- **JavaScript**: Utilizado para interação dinâmica e manipulação avançada do DOM.
- **Sass**: Empregado para criar uma interface responsiva e visualmente atraente.
- **HTML**: Estruturado com HTML5 para melhorar a acessibilidade e a semântica da web.

## Recursos

- **Comprimento da Senha Ajustável**: Os usuários podem ajustar o comprimento da senha usando um controle deslizante.
- **Cópia com um Clique**: Senhas podem ser copiadas com um simples clique no texto da senha gerada.
- **Verificação da Área de Transferência**: Inclui um campo de texto para verificar se a senha foi copiada com sucesso.

## Desafios e Aprendizados

Este projeto foi uma ótima experiência de aprendizado, pois envolveu manipulações intricadas do DOM e técnicas avançadas de CSS. Desenvolver o controle deslizante para ajustar o comprimento da senha me ensinou sobre elementos de design responsivo e manipulação de entradas do usuário em JavaScript. Implementar a funcionalidade da área de transferência foi desafiador, mas gratificante, pois melhorou significativamente a experiência do usuário da aplicação.
